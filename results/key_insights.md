# Key Insights

## 1. Value concentration supports prioritisation

Dapagliflozin and EpiPen represented 68.7% of total usage value in the selected portfolio. This supports prioritising monitoring effort around A-class medicines.

## 2. ABC classification is necessary but incomplete

ABC analysis ranks by usage value, but pharmacy inventory decisions also need clinical criticality, expiry risk, storage constraints, and seasonality.

## 3. Seasonal C-class products still need attention

Amoxicillin and Fexofenadine were lower usage-value products, but their demand patterns showed seasonal risk. This means C-class status should not automatically imply minimal planning.

## 4. Forecasting method should match demand pattern

Holt-Winters suits recurring seasonal peaks, while Simple Exponential Smoothing is more appropriate for stable repeat demand.

## 5. Service level is a clinical and operational metric

For pharmacy inventory, a stockout is not simply a lost sale. It can cause owed items, delayed treatment, emergency sourcing, and patient-safety risk.

## 6. Final operating principle

Clinical consequence, not cost alone, should drive inventory design.
