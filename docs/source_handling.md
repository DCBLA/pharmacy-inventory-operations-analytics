# Source Handling

The original source material was a group presentation file and academic assignment remit. Those files are not included in this portfolio repository.

Excluded materials:

- original slide deck
- raw slide screenshots
- assignment remit
- student identifiers
- module codes and submission details
- raw university-specific marking material
- full reference slides

The portfolio version keeps only the professional case-study content: business problem, analytical methods, key findings, recommendations, and limitations.

## Visual Treatment

Raw slide screenshots are excluded because the original deck contains student identifiers, university-specific submission details, local case details, and third-party source material. The public portfolio version uses redesigned summary visuals instead of screenshots of the submitted slides.

## Group Work Disclosure

This case study was adapted from a group presentation. The README identifies the work as a group presentation and states the individual leadership role separately. This avoids presenting group work as a solo project while still showing leadership, structure, and communication skills.
